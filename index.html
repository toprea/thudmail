thudmail

ruby webmail

Download this project as a .zip file Download this project as a tar.gz file

Overview

thudmail aims to be a gmail-like webmail system. It is not usable yet.

Features

  • threaded message view (conversations)
  • fast full-text search
  • labels
  • a contact list with intelligent autopopulation
  • MIME attachment support
  • receiving mail directly
  • polling external POP3/IMAP accounts for mail

No

  • POP3 or IMAP server support

Architecture

  • A Rails app providing a REST API around a message store
  • an HTML+JS client app that talks to that API
  • ferret for indexing (for now)
  • message metadata in database, message source in files on disk
  • polling external accounts (via a ruby library, or something like fetchmail)
  • direct delivery of primary-account mail (via a .forward pipe to an import script)

Installing

Only tested with Rails 3.2.1.

After downloading, create the db schema: rake db:create db:migrate

Sample Data

thudmail uses a small portion of the publicly-available Enron email dataset from http://www.cs.cmu.edu/~enron/

Import the sample data: cd sampledata ; ./import.sh

Run the development server: rails server and hit http://localhost:3000

Log in as "user1 / user1" or "user2 / user2"
